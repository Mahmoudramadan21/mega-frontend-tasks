## Challenge Overview
Build a clean, simple, and fully functional **Scientific Calculator** using **only pure Vanilla JavaScript** (no frameworks or libraries).

This project proves you have mastered **every single concept** from lessons **#019 to #039** of Elzero's famous course.

**Goal:** Keep it simple, clean, readable, and 100% working.

| Feature                         | Must Be Included |
