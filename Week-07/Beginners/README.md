## 🛠️ Required CSS Concepts (The Motion Challenge)

You must implement each of these concepts at least once in your project:

| Lesson        | Concept                                 | Implementation Idea                                                                                              |
| :------------ | --------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| **#65 - #68** | `2D Transforms`                         | Use `scale()` to enlarge a card on hover, and `translate()` to move elements smoothly.                           |
| **#69 - #70** | `Transform Origin`                      | Change the `transform-origin` of an icon (like a bell) so it swings from the top when rotated.                   |
| **#71 - #73** | `3D Transforms` & `backface-visibility` | **[Mandatory]** Create a 3D Flipping Product Card. Show the product image on the front, and details on the back. |
| **#74 - #76** | `@keyframes` & `animation` properties   | Create an entrance animation (e.g., elements fade in and slide up when the page loads) using `fill-mode`.        |
| **#77**       | `Practical Animations`                  | Build a custom continuous loading spinner or an up-and-down bouncing arrow using `infinite` iteration.           |

---

## 📝 Project Requirements Checklist

- [ ] **The 3D Flip Card:** Build at least one card that flips 180 degrees on hover. You must use `perspective`, `transform-style: preserve-3d`, and `backface-visibility: hidden`.
- [ ] **Interactive Hover Effects:** Add `transform: scale()` or `transform: translateY()` to your buttons or links to make them feel responsive and interactive.
- [ ] **Custom Loading Spinner:** Create a standalone CSS loading spinner using `@keyframes` and `animation-timing-function` (no GIFs allowed).
- [ ] **Entrance Animation:** Make your page content animate smoothly into view when the page loads (using `animation-delay` for different elements is a huge plus).
- [ ] **Transform Origin Adjustment:** Specifically modify the `transform-origin` of at least one element to create a unique rotation or scaling effect.
- [ ] **Smooth Transitions:** Ensure all transform hover effects use the `transition` property so they don't snap instantly.
